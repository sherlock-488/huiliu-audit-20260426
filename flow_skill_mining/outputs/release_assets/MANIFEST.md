# Flow Skill Mining Large Output Assets

The local workspace keeps the original UTF-8 JSONL outputs under `flow_skill_mining/outputs/`.

Several generated JSONL files exceed GitHub's single-file repository limit when uncompressed, so compressed copies are provided here for GitHub upload:

- `case_records.jsonl.gz` -> local `flow_skill_mining/outputs/case_records.jsonl`
- `session_trajectories.jsonl.gz` -> local `flow_skill_mining/outputs/session_trajectories.jsonl`
- `flow_states.jsonl.gz` -> local `flow_skill_mining/outputs/flow_states.jsonl`
- `train.jsonl.gz` -> local `flow_skill_mining/outputs/splits/train.jsonl`

All uncompressed source JSONL files are generated by rerunning the scripts documented in `flow_skill_mining/outputs/README_REPORT.md`.
