# 流程型 Skill 自进化挖掘总报告

## 1. 数据结构
这批数据是训练师精标 goodcase JSONL，每行包含 `sessionId`、`问题场景`、`source`、`input`、`target`。`input` 内含方案列表、当前时间、系统信号、商品/已选商品信息和对话历史；`target` 是 JSON 字符串，包含 `Response`、`ResponseSolution`、`DialogueAgreeSolution`。

## 2. 为什么适合流程 skill
它的核心监督信号是方案选择、确认态和信号约束，而不是话术风格。`ResponseSolution` 和 `DialogueAgreeSolution` 能直接对应流程动作、前置条件与执行确认。
