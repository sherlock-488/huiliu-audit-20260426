# 更新日志

## 2026-03-24 - 新增能力边界检查

### 新增功能

1. **退货/取回关键词过滤**
   - 智能客服没有退货取回能力，需要过滤相关回复
   - 关键词列表：
     - 退货、退回、取回
     - 保留商品、上门取件、取件
     - 自行处理、丢弃、扔掉、寄回、退还
     - 商品处理、物品处理、货物退回

2. **其他业务名称过滤**
   - 避免混淆其他业务线，需要过滤相关业务名称
   - 关键词列表：
     - 拼好饭、放心吃
     - 美团买菜、美团优选、美团闪购
     - 象大厨、象会员、象超市

### 修改文件

- `filters.py` - 新增 `check_return_keywords()` 和 `check_other_business_keywords()` 函数
- `process.py` - 在 `apply_filters()` 中应用新的过滤规则
- `SKILL.md` - 更新文档，添加能力边界检查说明

### 测试结果

所有测试用例通过：
- ✅ 正确识别包含退货/取回关键词的回复
- ✅ 正确识别包含其他业务名称的回复
- ✅ 正常回复不受影响

### 使用方法

无需额外配置，处理数据时会自动应用新的过滤规则：

```bash
python3 process.py input.txt output.jsonl
```

过滤统计中会显示：
- `包含退货/取回关键词` - 被过滤的数量
- `包含其他业务名称` - 被过滤的数量
